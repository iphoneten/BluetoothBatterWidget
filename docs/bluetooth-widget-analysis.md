# 蓝牙小组件分析与改进方向

## 市面常见能力

- 连接状态明确：已连接、未连接、蓝牙关闭、缺少权限需要区分，不能只靠电量图标表达。
- 设备识别：显示当前耳机或蓝牙音频设备名称，避免用户不知道小组件对应哪个设备。
- 电量表达双通道：图标展示大致电量，百分比文本展示精确电量。
- 断开态不误导：断开后不显示旧电量图标，避免用户误以为设备仍在线。
- 更新时间：显示最近刷新时间，降低用户对桌面刷新延迟的困惑。
- 点击刷新或打开设置：小组件点击后可以主动刷新；缺权限时打开应用授权。
- 紧凑外观：尺寸贴近系统卡片，圆角背景，信息层级清楚。

## 当前已落地

- 圆角白色卡片背景。
- 更紧凑的小组件高度和内部间距。
- 断开后隐藏电量图标。
- 点击小组件刷新，缺权限时打开主应用。
- 通过蓝牙 Profile 判断连接状态，降低缓存电量误判。
- 设备名、电量百分比和刷新时间展示。

## 后续可扩展

- 多设备列表：同时展示多个已连接蓝牙音频设备。
- 左右耳/耳机盒电量：需要设备或系统广播支持，不能对所有耳机保证。
- 多种小组件尺寸：提供 1x1 紧凑版和 2x1 详细版。
- 自定义样式：深色背景、透明背景、显示/隐藏设备名。
- 应用内预览：在主应用里展示与桌面一致的小组件预览。
